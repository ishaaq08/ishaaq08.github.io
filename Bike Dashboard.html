About the Data

The data collected information regarding how bike purchases vary across demographics. For example, are higher income earners more likely to purchase a bike then lower income earners, or what is the preferred distance to commute by bike. Some of the demographic data included :
  • Marital Status
  • Gender
  • Income Level
  • Number of Children
  • Occupation
  • Education
  • Home Owner

Methodology

The project was divided into 3 main areas:

1) Data Cleaning
  • Removed duplicates.
  • Formatted currencies to ensure consistency.
  • Utilised nested if statements to create age brackets which would aid visualisations - individual ages would create too many different categories, negatively effecting visualisations.
2) Pivot Tables
  • Created pivot tables and pivot charts which would later be organised into a dashboard format.
3) Dashboard
  • Created an interactive dashboard with the addition of filter buttons to provide interactivity for users.
