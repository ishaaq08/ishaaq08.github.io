About the Data

The dataset used was real survey data collected by Alex Freberg regarding career feedback from professionals in the data industry. Click here to view the data set. Examples of some of the data collected is as follows:

  • Location of respondent
  • Gender
  • Favourite programming language
  • Data job
  • Salary

Methodology

Data Transformation
Before building the dashboard Power Query, a pre-processing data engine a part of the Power BI package, was utilised to clean and transform the data. Data transformation is vital in the data analytics workflow as it enhances the sability of the data for analysis. The following steps were performed to transform the data:

  • Removed unused columns
  • Split columns by delimiter - ensure consistent formatting of data
  • Created custom columns whose values are computed from other columns
Data Visualisation
The data visualised along with the type of visual used are as follows:

  • 1) Number of respondents from each country - Treemap
  • 2) Favourite programming language - Stacked Column Chart
  • 3) Average salary by data job - Stacked Bar Chart
  • 4) Level of difficulty breaking into data - Donut Chart
  • 5) Happiness with work life balance - Gauge
  • 6) Happiness with current salary - Gauge
  • 7) Average respondent age - Card
  • 8) Total respondents - Card
