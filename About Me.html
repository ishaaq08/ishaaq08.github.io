My Journey to Coding and Data

Throughout my degree I utilised a variety of engineering software to conduct simulations and was always fascinated by the underlying workings of these software. It wasn't until my masters year in which I studied a module concerning the algorithms behind these simulations and utilised coding for the very first time - the language was C++ in a Linux OS.

I began self-learning Python and in my internship I was able to apply my coding skills on a real-world coding project. The project involved developing a Python API integrated with a specialist engineering tolerance analysis package. This was a substantial learning opportunity in which I enhanced my expertise in Python and Power BI. Since then I have been self-learning data analytics technologies such as Python, SQL, Power BI and Tableau through porfolio projects, Udemy courses and solving online tasks.

My interest is not limited to data analytics. Technology alone is very to interesting me and I enjoy constantly expanding my knowldege. For example, I built this portfolio webiste on the source code provided by Massively, however I self-learned various areas of HTML to adapt the code to my liking. I have also purchased a complete Linux training course on Udemy. My interest in Linux stems from my interest in the use of High Performance Computing (HPC) to solve multi-physics problems. Linux dominates the HPC domain and so gaining traction in this OS is vital if I wish to secure a role in HPC for engineering applications.
